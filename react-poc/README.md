Steps to set up react PoC
--------------
Prerequisite: nodeJS is installed

 * Download dependencies
```sh
npm run installAll
```

 * Run an HTTP server
```sh
npm start
```

Other scripts
--------------
```sh
npm run storybook
```

```sh
npm run buildBo
```
