Steps to set up openui5 PoC
--------------
Prerequisite: nodeJS is installed

 * Download dependencies
```sh
npm install
```
 * Run an HTTP server
```sh
node server.js
```
 * Access by link [http://localhost:9876/openui5-poc/public](http://localhost:9876/openui5-poc/public)
